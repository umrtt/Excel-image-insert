# Excel Image Insert - Python Project

**A production-quality desktop application for inserting images into Excel files**

## Features

✨ **Core Features:**
- Select Excel workbooks (.xlsx, .xls, .xlsm)
- Select one or multiple image files
- Choose target sheet and cell positions
- **NEW: Select cell ranges directly from Excel** (visual range picker)
- Auto-fit images to cell size or specify custom dimensions
- Handle merged cells properly
- Support cell ranges (e.g., A1:C5) for multi-cell placement
- Delete existing images before insertion
- Batch insert multiple images
- Real-time status and logging

🎨 **User Interface:**
- Modern, responsive HTML5/CSS3 frontend
- Real-time file picker dialogs
- **NEW: "Select Range" button for visual range selection**
- Image list with details and removal options
- Live logging with timestamps
- Status indicators

🏗️ **Architecture:**
- **Clean Architecture** with separated concerns
- **SOLID Principles** strictly followed
- **Dependency Injection** for testability
- **Object-Oriented Design** throughout
- Modular and maintainable codebase

## Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (Eel framework)
- **Backend:** Python 3.8+
- **Excel Handling:** xlwings
- **Image Processing:** Pillow
- **GUI Framework:** Eel (Electron-like wrapper)

- Python 3.8 or higher


```bash

```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
python main.py
```

The application will open in your default browser with a GUI window.

## Usage Guide

### Step 1: Select Excel File
1. Click "Browse" to open the file selection dialog
2. Select an Excel workbook (.xlsx, .xls, or .xlsm)
3. The available sheets will be displayed

### Step 2: Configure Settings
1. Select the target sheet from the dropdown
2. Select images by clicking "Add Images"
3. Choose one or multiple images in the file picker

### Step 3: Set Image Position
1. Enter the target cell position (e.g., A1, B2, C10)
2. Choose "Auto-fit" or manually specify dimensions

### Step 4: Insert Images
1. Click "Insert Images" button
2. Monitor the status log for progress
3. Images will be inserted and the file will be saved automatically

## Architecture Explanation

### SOLID Principles Applied

**1. Single Responsibility Principle (SRP)**
- Each class has one clear responsibility
- `Logger`: Only handles logging
- `Validator`: Only validates input
- `ExcelHandler`: Only handles Excel operations

**2. Open/Closed Principle (OCP)**
- System is open for extension (add new image formats)
- Closed for modification (existing code doesn't change)
- Use of abstract interfaces enables this

**3. Liskov Substitution Principle (LSP)**
- All implementations of `IExcelHandler` can be substituted
- All implementations of `IFileHandler` can be substituted
- Contracts are properly maintained

**4. Interface Segregation Principle (ISP)**
- Interfaces are small and specific
- `IExcelHandler` - Only Excel operations
- `IFileHandler` - Only file operations
- `ILogger` - Only logging operations

**5. Dependency Inversion Principle (DIP)**
- High-level modules depend on abstractions
- `ExcelService` depends on `IExcelHandler`, not `ExcelHandler`
- Services use constructor injection
- Easy to swap implementations for testing

## Advanced Features

### Batch Processing
Insert multiple images into multiple cells in a single operation:
```python
# Backend automatically handles batch operations
operations = [
    {'sheet_name': 'Sheet1', 'cell_position': 'A1', 'image_path': 'img1.png'},
    {'sheet_name': 'Sheet1', 'cell_position': 'B1', 'image_path': 'img2.png'},
]
```

### Merged Cell Support
The application automatically detects merged cells and fits images into the entire merged range.

### Auto-scaling
Images are automatically scaled to fit cell dimensions while maintaining aspect ratio.

### Error Handling
Comprehensive error handling with user-friendly messages and detailed logging.

### Logging
- Console logging with colored output
- File logging in `logs/debug_DDMM.txt` format
- Timestamps on all log entries
- Multiple log levels (DEBUG, INFO, WARNING, ERROR, CRITICAL)

## API Endpoints (Eel Exposed)

### File Operations
- `select_excel_file()` - Open Excel file
- `get_sheets()` - Get available sheets
- `close_excel_file()` - Close current file

### Image Operations
- `select_images()` - Select image files
- `add_image()` - Add image to list
- `remove_image()` - Remove image by index
- `clear_images()` - Clear all images
- `get_images()` - Get all images

### Insertion
- `insert_images()` - Insert all queued images
- `insert_single_image()` - Insert single image

### Status
- `get_status()` - Get application status

## Troubleshooting

### Excel File Won't Open
- Ensure the file is not corrupted
- Check that Excel is installed on your system
- Try closing the file in Excel first

### Images Not Appearing
- Verify image file format is supported (.png, .jpg, .gif, .bmp)
- Check that cell position is valid (e.g., A1, B2)
- Ensure sheet name is spelled correctly

### Port Already in Use
If port 8000 is in use, modify `main.py`:
```python
eel.start('index.html', port=8001)  # Use different port
```

### Library Import Errors
Reinstall dependencies:
```bash
pip install -r requirements.txt --force-reinstall
```
